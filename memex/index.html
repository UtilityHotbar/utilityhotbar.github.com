Instructions

            
Each line in this editor is similar to a cell in a Notebook.

            
Navigation

            
    
                
  • Ctrl+Up/Down - Switch between cells
    • 
                
  • Ctrl+Shift+Up - Collapse multiple empty cells into one
    • 
            

            
Cell Manipulation

            
    
                
  • Ctrl+Enter - Create new cell/split current cell (when caret is in the middle of a cell)
    • 
                
  • Ctrl+Backspace - Merge cell content with previous cell (when caret is at the start of a cell)
    • 
                
  • Ctrl+Shift+Enter - Explode a cell into a series of cells (one cell per newline)
    • 
                
  • Ctrl+Shift+Backspace - Collapse a series of cells into one (one cell per newline)
    • 
            

            
Document

            
    
                
  • Ctrl+S - Save to output (for copying)
